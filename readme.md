# Breadth-First Search in Chess Table

**Реализация алгоритма обхода графа в ширину (BFS) на языке C++ с использованием фреймворка Qt**

Это учебный/демонстрационный проект, который визуализирует работу алгоритма **BFS (Breadth-First Search)** на шахматной доске.  
Алгоритм обходит граф, где вершины — это клетки шахматной доски, а рёбра — возможные ходы выбранной шахматной фигуры (конь, ферзь или ладья).

## ✨ Возможности

- Графический интерфейс на Qt
- Визуализация шахматной доски
- Поддержка трёх фигур:
  - Конь (Knight)
  - Ферзь (Queen)
  - Ладья (Rook)
- Пошаговая или анимированная демонстрация работы BFS
- Подсветка порядка обхода клеток
- Выбор стартовой позиции

## 🛠 Технологии

- **C++**
- **Qt 6** (Widgets + Qt Designer)
- **CMake** — система сборки
